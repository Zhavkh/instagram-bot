# Instagram Follower Bot

Автоматический бот для увеличения подписчиков в Instagram через Follow/Unfollow стратегию.

## Возможности

- ✅ Автоматические подписки/отписки
- ✅ Работает 24/7 на Render.com (бесплатно, БЕЗ карты)
- ✅ Управление через Telegram
- ✅ Защита существующих подписчиков/подписок
- ✅ 3 режима работы (Безопасный/Умеренный/Агрессивный)
- ✅ Автоматический режим по расписанию
